プライバシーポリシー

「こっこちゃんとあそぼ」運営者（以下「運営者」といいます）は、本ウェブサイトおよび関連アプリケーション（以下「本サービス」といいます）におけるユーザーの個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

第1条（個人情報の定義）

本ポリシーにおける「個人情報」とは、個人情報保護法に定める個人情報、すなわち生存する個人に関する情報であって、当該情報に含まれる氏名、生年月日、その他の記述等により特定の個人を識別することができるものを指します。

第2条（取得する情報）

運営者は、本サービスの提供にあたり、以下の情報を取得することがあります。

  • SNSアカウント情報（TikTok、YouTube、X、note 等）：投稿機能を利用する際に、各プラットフォームのAPIを通じて取得するアカウントID・表示名・公開プロフィール等
  • OAuth認証情報：各SNSプラットフォームから付与されるアクセストークン・リフレッシュトークン
  • アクセスログ：本ウェブサイトへのアクセス日時、IPアドレス、ブラウザ種別等

第3条（利用目的）

運営者は、取得した情報を以下の目的のために利用します。

  • 本サービスの提供および運営のため
  • 各SNSプラットフォームへのコンテンツ自動投稿機能を提供するため
  • ユーザーからのお問い合わせに対応するため
  • 本サービスの不正利用を防止するため
  • 本サービスの改善および新機能の開発のため

第4条（第三者提供）

運営者は、以下に掲げる場合を除き、ユーザーの同意なく取得した情報を第三者に提供することはありません。

  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
  • 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合

第5条（外部APIの利用）

本サービスは、以下の外部プラットフォームのAPIを利用します。各APIの利用にあたっては、それぞれのプラットフォームが定めるプライバシーポリシーが適用されます。

  • TikTok for Developers（プライバシーポリシー）
  • YouTube Data API v3 / Google API（プライバシーポリシー）
  • X (Twitter) API（プライバシーポリシー）
  • note API

第6条（情報の保存と削除）

取得した認証トークン等の情報は、本サービス運営者のローカル環境においてのみ保存され、外部サーバーへの送信や第三者との共有は行いません。ユーザーはいつでも各SNSプラットフォームの設定画面から本アプリの認可を取り消すことができ、その場合、運営者は速やかに該当情報を削除します。

第7条（Cookie等の利用）

本ウェブサイトでは、ユーザーの利便性向上のためCookieを使用する場合があります。ユーザーはブラウザの設定によりCookieの受け入れを拒否することができます。

第8条（個人情報の開示・訂正・削除）

ユーザーは、運営者に対して自己の個人情報の開示・訂正・削除を求めることができます。お申し出があった場合、運営者は本人確認を行ったうえ、合理的な期間内に対応いたします。

第9条（プライバシーポリシーの変更）

運営者は、必要に応じて本ポリシーを変更することがあります。変更後の本ポリシーは、本ウェブサイトに掲示した時点から効力を生じるものとします。

第10条（お問い合わせ窓口）

本ポリシーに関するお問い合わせは、X（Twitter）@kokkotoasobo のDMまでお願いいたします。
